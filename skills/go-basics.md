# Go Basics

## Variables

```go
var name string = "Samad"
```

## Functions

```go
func greet(name string) string {
    return "Hello, " + name
}
```

## Notes

- Go is a compiled language
- No try-catch, errors as return values
- Use `:=` for short variable declaration
